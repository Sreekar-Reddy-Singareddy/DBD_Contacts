--------------- Server & DB Config --------------

1. Node JS server must be hosted on a computer first.
2. Server Port: 3000
3. MySQL server IP address: Change this in the Server.js
4. Database Name: DBD_CLASS
5. MySQL server username and password: Change them in the Server.js
6. Once server is hosted, the server IP address must be updated in the Android app. 
   Change this in just one file "RetrofitService.java".
7. Read_Data.js is not a part of the application. It is used to read "contact.csv" file 
   into the MySQL database tables.
8. The data from MySQL is exported to "db_data.sql" file. Import this to stay 
   consistent with the schema. Else the application might not work as expected.
